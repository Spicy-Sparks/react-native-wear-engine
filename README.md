# react-native-wear-engine

React Native Huawei Wear Engine

## Installation

```sh
npm install react-native-wear-engine
```

## Usage

```js
import WearEngine from "react-native-wear-engine";

// ...

const result = await WearEngine.multiply(3, 7);
```
